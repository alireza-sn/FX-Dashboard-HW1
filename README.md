# FX Dashboard - Professional Analytics Portal

## Description
The **FX Dashboard** is a high-performance analytics portal designed for International Forex Brokerage professionals. It leverages **Apple’s Human Interface Guidelines** to provide a premium, intuitive, and highly efficient user experience for monitoring market trends, managing clients, and tracking financial performance.

## Key Features
*   **iCloud-Style Login:** A secure, centered authentication interface with a sophisticated frosted-glass aesthetic.
*   **Liquid Glass Design:** Premium UI elements utilizing `backdrop-filter` blur effects and translucent layers for a modern, high-end feel.
*   **Collapsible Sidebar:** A space-efficient navigation system that supports both full and icon-only modes with state persistence via `localStorage`.
*   **Intelligent Dark Mode:** A seamless theme engine that adapts the entire interface to professional light or high-contrast dark environments.
*   **Responsive Fluid Grid:** A robust 12-column grid system that automatically adapts layout components for Mobile, Tablet, and Desktop displays.
*   **Live Analytics:** Interactive data visualizations powered by Chart.js, featuring real-time market signals and system health monitoring.

## Tech Stack
*   **Core:** HTML5, CSS3 (Custom Variables, Flexbox, Grid), Modern JavaScript (ES6+).
*   **Libraries:** [Chart.js](https://www.chartjs.org/) for interactive financial data visualization.
*   **Icons:** Minimalist SVG-based icon set with consistent stroke weighting.
